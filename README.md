# 🏎️ Sonny Angel F1 Series - React მინი-თამაში

ეს არის React-ზე აწყობილი ვებ-აპლიკაცია (მინი-თამაში), რომელიც აერთიანებს ფორმულა 1-ის თემატიკას და "Sonny Angel"-ის საკოლექციო ფიგურებს.

მომხმარებელი ირჩევს სასურველ მრბოლელს, ტრასას და რბოლის ხანგრძლივობას, იწყებს რბოლას რეალურ დროში განახლებადი ამინდითა და ტელემეტრიით, ხოლო რბოლის დასრულებისას ხსნის ახალ ფიგურას, რომელიც ინახება მის გარაჟში.

---

## 📋 შესრულებული მოთხოვნები & ტექნოლოგიები

- **React & Hooks:** აწყობილია ფუნქციონალური კომპონენტებით, `useState`, `useEffect`, `useMemo`, `useCallback` და custom ჰუკებით.
- **4 გვერდი (React Router v6):** `Home` (ლობი), `Game` (რბოლა), `Standings` (ლიდერბორდი) და `Collection` (გარაჟი).
- **API (Axios):** ინტეგრირებულია **Jolpica F1** (standings) და **Open-Meteo** (ცოცხალი ამინდი ტრასებზე).
- **LocalStorage:** ინახავს მომხმარებლის კოლექციას, რბოლების ისტორიას, აქტიურ თემას და ენას.
- **ანიმაციები & მოდალები (Framer Motion):** საწყისი შუქნიშნების, პაკეტის გახსნის ანიმაცია და custom `Modal` კომპონენტი.
- **რესპონსივი:** სრულად ერგება მობილურებს, ტაბლეტებსა და დესკტოპებს.
- **SCSS Modules (SASS):** მოდულარული სტილები დიზაინ სისტემით (ბონუსი).
- **Dark / Light Mode:** მუქი/ღია რეჟიმები, ინახება ბრაუზერში (ბონუსი).
- **ორენოვნება (ქართული / English):** თარგმანების კონტექსტი ენების გადასართველად (ბონუსი).

---

## ▶️ ლოკალურად გაშვების ინსტრუქცია

პროექტის ლოკალურად გასაშვებად ჩაწერეთ ტერმინალში:

1. **პაკეტების ინსტალაცია:**
   ```bash
   npm install
   ```

2. **დეველოპმენტ სერვერის გაშვება:**
   ```bash
   npm run dev
   ```
   *ამის შემდეგ ბრაუზერში გახსენით: `http://localhost:5173/`*

3. **პროდაქშენ ბილდის აწყობა:**
   ```bash
   npm run build
   ```
